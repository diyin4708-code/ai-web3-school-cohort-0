# 个人学习计划

## 总体路径

从 **AI × Web3 Bridge** 直接切入，重点打通 Agent ↔ 链上交互的完整链路。

## 每日流程

1. 阅读当日课程/Handbook 章节
2. 写 `daily/YYYY-MM-DD.md` 笔记
3. 生成打卡草稿 → 手动提交 WCB
4. 遇到问题 → 记入 `handbook-feedback/`
