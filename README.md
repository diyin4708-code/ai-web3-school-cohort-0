# AI × Web3 School — Cohort 0

个人学习日志与 Proof-of-Work 仓库。

## 关于

[AI × Web3 School](https://aiweb3.school/zh/handbook/) 是一个面向 builders 的开源学习计划，由 [LXDAO](https://lxdao.io/) 与 [ETHPanda](https://ethpanda.org/) 共同发起。

这个仓库记录我在 AI × Web3 School 的学习过程、每日打卡、实验项目、任务记录和 Handbook 反馈。
